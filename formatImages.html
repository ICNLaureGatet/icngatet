Formats d'images

Nous allons dans cette partie nous intéresser à 5 formats de fichiers image (même s'il en existe bien d'autres) :

  • BMP : Bitmap, format utilisé par Windows ;
  • TIFF : Tagged Image File ;
  • JPEG ou JPG : Joint Photographic Expert Group ;
  • GIF : Graphics Interchange Format ;
  • PNG : Portable Network Graphic.
Il convient de choisir judicieusement le format d'une image suivant l'usage que nous souhaitons en faire.

Certains logiciels (comme The GIMP) permettent d'ouvrir tous les fichiers images aux formats cités ci-dessus et de les convertir d'un format à l'autre. Il est ainsi possible de comparer des images qu'on modifieraient d'un format à l'autre.

En PNG et JPG, un niveau de compression est demandé. Plus il est élevé, plus le poids de l'image est faible.
En JPG, on perd également en qualité (même si ce n'est pas vraiment perceptible à l'œil nu - tout dépend de la nature de l'image).

BMP et TIFF sont des formats très lourds : les fichiers ne sont pas compressés, la qualité est donc meilleure.
Le format TIFF est de plus plutôt orienté vers les professionnels.

Le JPEG est un format de compression maximale de nos jours. Ce format est de ce fait très utilisé sur Internet. Par contre cela altère la qualité de l'image, la compression se faisant avec perte d'information.
Ce format est principalement utilisé pour des photos. Avec des graphiques, on peut voir l'image baver.

Le GIF est un format de compression sans perte de données. Il limite le nombre de couleurs de chaque calque (256), ce qui le rend moins pratique sur des photos ou sur des images avec des dégradés importants de couleurs.
L'algorithme de compression étant un logiciel propriétaire, les utilisateurs du format GIF doivent payer une redevance à la société détentrice des droits.
Le GIF permet aussi la création d'animation (GIF animé).

Le PNG est un format libre qui devient le standard sur Internet. C'est également un format de compression (sans perte), mais avec un nombre plus important de couleurs que le GIF. Il supporte également la transparence. Il en existe deux versions (8 bits pour coder 256 couleurs et 24 bits pour plus de 16 millions de couleurs).

Il existe un format adapté à chaque image :

  • une photo : le JPG pour une utilisation classique ou sur Internet, le TIFF pour une utilisation professionnelle de grande qualité ;
  • un graphique avec peu de couleurs (moins de 256) : un PNG 8 bits ou un GIF ;
  • un graphique avec beaucoup de couleurs : un PNG 24 bits ;
  • une image animée : un GIF animé.
