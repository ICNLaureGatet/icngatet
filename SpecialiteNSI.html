ENSEIGNEMENT DE SPÉCIALITÉ
PREMIÈRE

Numérique et Sciences Informatiques (N.S.I.)

Descriptif

L'enseignement de spécialité Numérique et Sciences Informatiques s'appuie sur quatre concepts et la variété de leur interaction :

  • les données (textes, images, sons, mesures physiques, …) sous forme numérique ;
  • les algorithmes ;
  • les langages (python 3 sera privilégié) ;
  • les machines et leurs systèmes d'exploitation.
L'élève saura rechercher de l'information, partager des ressources en se souciant de l'usage responsable et critique de cette information.

Démarche de projet

Cet enseignement a vocation à multiplier les occasions de mise en activité des élèves sous forme variées : autonomie, esprit d'initiative mais aussi coopération au sein d'une équipe dans le cadre d'un projet.

Les élèves devront s'approprier les concepts en développant des projets conduits par des groupes de deux – trois élèves. Ces projets peuvent porter sur des problématiques issues des autres disciplines : simulation d'une expérience, travail sur des données socioéconomiques, sur un objet connecté, résolution d'un problème de traitement d'image, développement d'un site web avec gestion d'une base de données, …

Des points d'étapes destinés à valider des éléments, à contrôler l'avancement du projet ou à adapter les objectifs doivent être faits régulièrement avec le professeur.

Éléments du programme

Le programme est organisé autour de huit rubriques :

  • histoire de l'informatique ;
  • représentation des données : types et valeurs (entiers, flottants, caractères et chaînes de caractères) ;
  • représentation des données : types construits (tuples, listes ou tableaux, dictionnaires) ;
  • traitement de données en tables : traitement d'un fichier csv, recherche, tri, fusion, …
  • interactions entre l'homme et la machine sur le Web : examen du code HTML, événements sur des composants, formulaire d'une page Web ;
  • architectures matérielles et systèmes d'exploitation : transmission de données dans un réseau, interface homme-machine (IMH) ;
  • langage et programmation : construction d'un programme, jeux de tests ;
  • algorithmique : algorithmes classiques sur un tableau de données, et étude de leur coût, algorithme plus complexes.
